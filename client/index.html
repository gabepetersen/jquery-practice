Event Types

.on('event', callback(event) {...})

.on() event binder is currently the way to bind events to selected elements. All that is needed is to pass the type of event and use the callback to change things. Since JQuery 3.0, all shorthand events are deprecated :(


To the right we see that the 'click', 'mouseenter', 'mouseleave', 'resize', 'scroll', and 'keypress' events are being used


Note 1: Instead of keypress, you can also use 'keyup' or 'keydown' for specific button presses


Note 2: to access the data for some event callbacks, you can find a lot of the data in event.originalEvent


Note 3: There are also document events: 'load' and 'ready' that inform about when assets are loaded
